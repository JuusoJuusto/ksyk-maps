# 🔨 KSYK Builder - Complete Guide

## ✅ BUILDER IS NOW IN THE MAIN APP!

You can now access the full KSYK Builder directly from the home page - no need to navigate to a separate page!

---

## 🎯 How to Access

1. Open KSYK Maps home page
2. Look for the tabs at the top-left: **Map | Builder | Schedule | Settings**
3. Click the **"Builder"** tab (hammer icon 🔨)
4. Start creating!

---

## 🏗️ What You Can Build

### 1. Buildings
- Custom shapes (click points to draw)
- Rectangle mode (click two corners)
- Set building name, color, floors
- Position on campus map

### 2. Rooms
- Draw room shapes
- Set room number (e.g., M12, K15)
- Choose room type (classroom, lab, office, etc.)
- Set capacity and floor
- Add to existing buildings

### 3. Hallways
- Connect rooms together
- Enable navigation between spaces
- Set width and floor
- Create walking paths

### 4. Stairways & Elevators
- Connect different floors
- Enable vertical navigation
- Required for multi-floor routing

### 5. Doors
- Mark entry/exit points
- Connect rooms to hallways

---

## 📝 Step-by-Step: Creating a Building

1. **Click "Builder" Tab**
   - Switch from Map view to Builder

2. **Select "Building" Tool**
   - Click the Building button in the left sidebar

3. **Fill in Building Details**
   - Name: Building code (e.g., "M", "K", "L")
   - English Name: Full name (e.g., "Music Building")
   - Finnish Name: Finnish translation
   - Floors: Select which floors (1, 2, 3, etc.)
   - Color: Choose building color

4. **Choose Shape Mode**
   - **Rectangle**: Click two corners (fast!)
   - **Custom**: Click multiple points for complex shapes

5. **Start Drawing**
   - Click "Start Drawing" button
   - Click on the map to place points
   - For rectangle: Click start corner, then end corner
   - For custom: Click each corner point

6. **Finish Building**
   - Click "Finish" button
   - Or click near first point to close shape
   - Building is saved automatically!

7. **View Your Building**
   - Switch back to "Map" tab
   - See your building on the campus map!

---

## 📝 Step-by-Step: Adding Rooms

1. **Select "Room" Tool**
   - Click the Room button in sidebar

2. **Choose Building**
   - Select which building from dropdown

3. **Fill in Room Details**
   - Room Number: e.g., "M12"
   - English Name: e.g., "Music Room 12"
   - Finnish Name: e.g., "Musiikkiluokka 12"
   - Floor: Which floor (1, 2, 3)
   - Capacity: Number of seats
   - Type: classroom, lab, office, etc.

4. **Draw Room Shape**
   - Click "Start Drawing"
   - Draw rectangle or custom shape
   - Click "Finish"

5. **Room is Created!**
   - Appears on map
   - Searchable by room number
   - Available for navigation

---

## 🚶 Step-by-Step: Adding Hallways

**Important:** Hallways are required for navigation to work!

1. **Select "Hallway" Tool**
   - Click the Hallway button

2. **Choose Building**
   - Select building from dropdown

3. **Fill in Details**
   - Name: e.g., "Main Hallway"
   - Floor: Which floor
   - Width: 2-10 meters

4. **Draw Hallway Path**
   - Click "Start Drawing"
   - Draw the hallway route
   - Connect rooms together
   - Click "Finish"

5. **Navigation Enabled!**
   - Rooms connected by hallway can now be navigated
   - Path will show in navigation

## ⌨️ Keyboard Shortcuts

- **Delete**: Delete selected building
- **Ctrl+C**: Copy selected building
- **Ctrl+V**: Paste copied building
- **Escape**: Cancel current drawing
- **G**: Toggle grid on/off
- **S**: Toggle snap to grid

## 💡 Pro Tips

1. **Start with Buildings**
   - Create all buildings first
   - Then add rooms to each building
   - Finally connect with hallways

2. **Use Rectangle Mode**
   - Faster for regular shapes
   - Perfect for most rooms
   - Two clicks and done!

3. **Enable Grid Snap**
   - Keeps everything aligned
   - Professional-looking layouts
   - Press 'G' to toggle

4. **Add Hallways for Navigation**
   - Navigation requires hallways
   - Connect all rooms you want to navigate between
   - Add stairways for multi-floor navigation

5. **Use Consistent Colors**
   - Same color for related buildings
   - Makes campus map easier to read
   - Color-code by department

6. **Test Navigation**
   - Switch to Map tab
   - Click "Get Directions"
   - Test routes between rooms
   - Add missing hallways if needed

7. **Save Often**
   - Each shape saves automatically
   - But refresh page to see updates
   - Keep browser tab open while building

## 🐛 Troubleshooting

### "No route found" in navigation?
- **Solution**: Add hallways connecting the rooms
- Rooms need hallway connections to navigate
- Add stairways for different floors

### Building not appearing on map?
- **Solution**: Refresh the page
- Switch to Map tab
- Check if building was saved (look in Buildings list)

### Can't click "Finish" button?
- **Solution**: Need minimum 3 points for custom shapes
- For rectangle: Need 2 points (start and end corners)
- Make sure you've placed enough points

### Drawing disappeared?
- **Solution**: You may have cancelled by accident
- Press Escape cancels current drawing
- Click "Start Drawing" again

### Map is too zoomed in/out?
- **Solution**: Use zoom controls (top-right)
- Or press Ctrl+Scroll to zoom
- Click reset button to center view

## 📊 What Gets Saved

When you create items in the Builder:

- ✅ Buildings → Saved to Firebase database
- ✅ Rooms → Saved to Firebase database
- ✅ Hallways → Saved to Firebase database
- ✅ Custom shapes → Saved in building description
- ✅ All properties → Saved (name, color, floor, etc.)

Everything is persistent and will appear for all users!
